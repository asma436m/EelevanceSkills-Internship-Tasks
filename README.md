Project Twitter Engagement Analytics Dashboard using Power BI
This project is about looking at Twitter data to see how well tweets do and how people interact with them using Power BI. I cleaned up the data by fixing the date and time and making new columns to help with the analysis. I looked at things like how many likes and replies and retweets a tweet gets and how many people see it and what the engagement rate is. I used charts and graphs to help us understand this data.
Made some rules to filter out tweets like when they were posted what day of the week it was, how long the tweet was and some other things. I also looked at tweets that got a lot of engagement when they had pictures or videos. Compared these tweets to others to see which ones did better. I also wanted to see if tweets that got likes and replies did better than tweets that did not.
The Twitter Engagement Analytics Dashboard we made with Power BI is really cool because it lets users play around with the data and see what they can find. They can look at the picture and see what is trending or they can zoom in and see the details. The dashboard has lots of things, like charts and buttons that make it easy to use and understand the Twitter engagement data.
